# 🚀 EasyBrain 개발 로드맵

**0. 🎨 Design & Data (User Lead)**
* [ ] **GUI 고도화:** Toss 증권 스타일의 Borderless UI 및 디테일 폴리싱.
* [ ] **Data Pipeline:** PostHog (또는 GA) 연동하여 Session Replay 및 사용자 패턴 분석.
* [ ] **User Research:** 사용자 행동 데이터 기반 UX 개선.

**1. ✍️ Phase 1: Rich Editor (작성 경험)**
* [ ] **슬래시 커맨드 (`/`):** 메뉴 호출, 제목/할일/인용구 등 블록 변환 기능.
* [ ] **이미지 업로드 (Storage):** Supabase Storage 연동, 실제 파일 업로드 및 Drag & Drop.

**2. 🧠 Phase 2: Connectivity (연결성)**
* [ ] **Command Palette (`Cmd + K`):** 마우스 없이 페이지 이동 및 검색.
* [ ] **Backlinks & Graph:** 페이지 간 연결 관계 시각화.

**3. 🤖 Phase 3: AI Intelligence (지능)**
* [ ] **AI 자동 요약:** 작성된 메모 요약 및 자동 태깅.
* [ ] **RAG Chat:** "내 메모와 대화하기" 기능 구현.

**4. 📦 Phase 4: Productization (제품화)**
* [ ] **Auth:** 로그인/회원가입 (Supabase Auth).
* [ ] **Theme:** 다크 모드 및 모바일 반응형 최적화.
